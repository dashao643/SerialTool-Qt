# Qt串口工具

![](./screenshot/interface.png)

## 功能介绍
一个基于 Qt6 开发的通用串口工具，支持自定义数据发送，和功能按钮封装

## 版本更新记录
### v1.0.0(2026-04-18)
- 初始版本发布

### v1.0.1(2026-04-22)
- 新增：显示接收到的字节数
- 新增：自定义发送项插入功能

### v1.0.2(2026-05-01)

- 新增：选中计算数据量
- 优化：收到数据自动移动最下

## 编译环境

Qt6.10.1 + MSVC2022 + Windows10 64位
