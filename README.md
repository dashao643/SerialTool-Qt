# Qt串口工具

## 功能介绍
一个基于 Qt6 开发的通用串口、网络调试工具，支持自定义数据发送、IAP升级。

## 源码编译

```bash
# 配置 CMakePresets.json 中 qt6 安装路径后
cmake --preset release					# 创建build目录并配置项目
cmake --build build/release -j 16		# 并行构建项目
cd build/release && ./SerialTool.exe	# 执行exe文件
```
